# The Invisible Pulse 

**Translating Plant Biodata into Generative Art.**
*A Devine Passion Project by Muhammed Godil.*

## About
The Invisible Pulse is an interactive installation that measures the electrical conductivity of living plants and translates that data into real-time projection mapping visuals. The project explores the intersection of organic life and digital synthesis.

## Tech Stack
* **Hardware:** Arduino ESP32, Custom Galvanic Conductance Circuit.
* **Software:** TouchDesigner (Visuals), Ableton Live (Sound).
* **Technique:** Projection Mapping, Generative Algorithms.
